# Shooting Game.

A Javascript game shooting game. 
Live Demo: andre0n.github.io/red-circle/

## Run Locally

Clone the project

```bash
git clone https://github.com/Andre0n/JS-game.git
```

Go to the project directory

```bash
cd JS-game
```

Install dependencies

```bash
  yarn
```

Start the server

```bash
  yarn dev
```

## Deployment

To deploy this project run

```bash
  yarn build
```
